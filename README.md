# 🛒 Online Retail Sales Analysis

An end-to-end data analytics project that analyzes online retail sales data using Python, MySQL, and Power BI.

## 📌 Project Overview

This project focuses on analyzing online retail sales data to identify sales trends, top-performing products, customer behavior, and business performance.

The project follows an end-to-end data analytics workflow:

Python → MySQL → Power BI

## 🛠️ Tools & Technologies

- Python
- Pandas
- Jupyter Notebook
- MySQL
- SQL
- Power BI
- DAX

## 🔄 Project Workflow

### 1. Python – Data Cleaning & Preparation

Python was used to clean and prepare the raw retail sales dataset.

Key tasks:
- Data cleaning
- Handling missing values
- Removing invalid records
- Creating calculated Sales column
- Data transformation
- Preparing data for SQL analysis

### 2. MySQL – Data Analysis

The cleaned data was imported into MySQL for analysis.

SQL was used to analyze:
- Total Sales
- Total Orders
- Total Customers
- Total Quantity
- Monthly Sales
- Sales by Country
- Top 10 Products by Sales
- Top 10 Products by Quantity
- Top 10 Customers by Sales
- Customer Analysis

### 3. Power BI – Dashboard & Visualization

Power BI was used to create an interactive dashboard for business analysis.

The dashboard includes:

#### 📊 Sales Analysis
- Total Sales
- Total Orders
- Total Customers
- Total Quantity
- Monthly Sales Trend

#### 📦 Product Analysis
- Top 10 Products by Sales
- Top 10 Products by Quantity
- Product Performance

#### 👥 Customer Analysis
- Top 10 Customers by Sales
- Average Order Value
- Average Sales per Customer
- Customer Purchase Frequency
- Customer Value Segmentation

## 📈 Key Business Insights

- Identified top-performing products based on sales and quantity.
- Analyzed monthly sales trends to understand business performance.
- Identified high-value customers contributing significantly to revenue.
- Analyzed customer purchasing behavior.
- Compared sales performance across different countries.
- Created an interactive Power BI dashboard for business decision-making.
